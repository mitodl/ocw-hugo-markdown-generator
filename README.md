# ocw-to-hugo

## Installation

```shell
$ cd ~/path/to/temp/dir
$ git clone git@github.com:mitodl/ocw-to-hugo.git
$ cd ocw-to-hugo
$ npm install -g .
```

## Usage

```shell
$ ocw-to-hugo -s <source directory> -d <destination directory>
```

## Arguments

| Argument                | Valid values  |
| :-------------------- | :------------ |
| `s`        | `/path/to/open-learning-course-data` |
| `d`                  | `/path/to/hugo-markdown-output` |
